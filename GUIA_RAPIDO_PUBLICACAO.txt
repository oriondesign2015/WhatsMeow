┌──────────────────────────────────────────────────────────────────────────────┐
│ 🔧 SOLUÇÃO DE PROBLEMAS                                                      │
└──────────────────────────────────────────────────────────────────────────────┘

PROBLEMA: "Permission denied" ao fazer push
SOLUÇÃO:
  - Use token pessoal como senha (não a senha da conta)
  - Crie token em: https://github.com/settings/tokens
  - Marque permissões: "repo" e "workflow"


PROBLEMA: "remote: Repository not found"
SOLUÇÃO:
  - Verifique se você criou o fork no GitHub
  - Vá para: https://github.com/tulir/whatsmeow → Fork
  - Certifique-se que o remote está correto: git remote -v


PROBLEMA: "fatal: refusing to merge unrelated histories"
SOLUÇÃO:
  - git pull origin main --allow-unrelated-histories
  - Ou apague a pasta .git e comece de novo


PROBLEMA: "Your branch is ahead of 'origin/main'"
SOLUÇÃO:
  - Isso é normal! Significa que você tem commits locais
  - Faça push: git push origin main
